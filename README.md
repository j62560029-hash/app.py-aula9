# 👁️ Simulador de Visão Computacional (Abordagem Baseada em Regras)

Este projeto é uma aplicação web interativa desenvolvida em Python com a biblioteca **Streamlit**. Ele foi criado especificamente para fins acadêmicos como uma simulação didática de um sistema de Visão Computacional para detecção de objetos (como pessoas, carros, animais e objetos cotidianos), operando de forma totalmente estática e leve, sem a necessidade de modelos pesados de Machine Learning ou Deep Learning.

## 📌 Problemática e Objetivo

Modelos reais de Visão Computacional (como YOLO ou ResNet) exigem alto processamento computacional (GPUs), dependências complexas e consomem muita memória, tornando o deploy inicial um desafio para estudantes e servidores gratuitos.

**O Objetivo deste projeto é:**
* Demonstrar a manipulação de matrizes de imagens utilizando a biblioteca **Pillow (PIL)**.
* Simular o comportamento visual de uma IA (tempo de resposta, bounding boxes e cálculo de acurácia fictício).
* Apresentar uma arquitetura de microsserviço web simples, ideal para validações de lógica e interfaces acadêmicas.

---

## 🛠️ Tecnologias Utilizadas

* **Python 3.10+**
* **Streamlit** (Para a interface gráfica web)
* **Pillow (PIL)** (Para o processamento e desenho nas imagens)
